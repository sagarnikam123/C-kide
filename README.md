- for windows user,open file in [Dev C++](http://www.bloodshed.net/devcpp.html) / [CodeBlocks](http://www.codeblocks.org/), compile & Run.
- for Linux user, comment below two line, like

	```C

	// getch();
	```

	compile & run as

	```sh

	```
